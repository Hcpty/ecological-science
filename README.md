# Readme
Paths and Ruts.

### 路与辙
能够到达目的地的途径称为路，路最明显的特征是上面的车辙。

### 天和地
万物相生相灭，基于这个条件，这个万物系统可以不需要天和地的外在干预而运转自如，这种治理办法称为无为而治。

### 生或死
生存下来了的基因生存了下来，没有生存下来的基因没有生存下来。
