# Readme
My philosophy.

第一

關子尹：從「折」引出的「哲」字，可引申為「斷疑」的意思，涉及的應用，從理論層面看，就是概念的區別，從實踐上看，則是行為的抉擇。

第二

爱因斯坦：所有困难的问题，答案都在另外一个层次上，同一层次的问题，很难靠同一层次的思考来解决。

人要先领悟至道，然后做人。
