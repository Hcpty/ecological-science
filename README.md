# Readme
Ruts and Path.

### 辙和道
古代道路最明显的特征就是上面有车辙，地点序列称为道路。知其然称为有辙，知其所以然称为知道。

### 天地无为而治
万物相生相灭，基于这个条件，这个万物系统可以不需要天地的外在干预而运转自如，这种治理方法称为无为而治。

### 万物并作有律
按照这个系统的运转的规律运作的元素会在这个系统中存续。
