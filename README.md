# Readme
My philosophy.

第一

關子尹：從「折」引出的「哲」字，可引申為「斷疑」的意思，涉及的應用，從理論層面看，就是概念的區別，從實踐上看，則是行為的抉擇。

第二

哲学这东西，平安时没甚鸟用，而当人福、祸临门的时候，就有非常大的用处了，兴盛时遵守正确的哲学，可以避免陷入梦幻主义，衰败时遵守正确的哲学，可以避免陷入虚无主义。
