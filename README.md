# Philosophy

## 道路和车轨
地点序列称为道路。先秦时期，车辆轮距不统一，一条道路上经常同时存在多条不重合的车轨。

## 天地无为而治
万物相生相克，基于这个条件，万物系统可以不需要天地的外在干预而运转自如，这种治理方法称为无为而治。

## 万物随性而行
万物如倒骑牛而行，牛导物行。
