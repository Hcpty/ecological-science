# 哲学

## 道路和车辙
地点序列称为道路，一条道路上经常同时存在许多车辙。

## 天地无为
万物相生相克，基于这个条件，万物系统不需要天地的外在干预就能运转自如，这种治理方法称为无为。无为是天地的道路，尽管各有各的车辙。

## 万物从性而为
万物如同倒骑牛而行，牛导物行。从性而为是万物的道路，尽管各有各的车辙。
