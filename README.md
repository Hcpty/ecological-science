### 我的哲学

關子尹：“從「折」引出的「哲」字，可引申為「斷疑」的意思，涉及的應用，從理論層面看，就是概念的區別，從實踐上看，則是行為的抉擇。”

个体和环境之间的相互作用共同决定了个体和环境的下一个状态……如此循环，循环体可以表示成一个反应方程式：

```个体的当前状态 + 环境的当前状态 -> 个体的目标状态 + 环境的目标状态```

哲学研究的内容是掌握各种反应物的性质及反应方程式。哲学实践的内容是借助反应改变反应物的状态。
